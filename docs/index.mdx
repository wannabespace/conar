Welcome to Conar — the AI-powered database client built for developers and data teams who want to move faster. Conar lets you connect to your PostgreSQL, MySQL, MSSQL, and ClickHouse databases from a sleek interface available on the web, desktop, and command line. Whether you're exploring data, debugging a query, or trying to understand a schema you've never seen before, Conar's built-in AI assistant is right there with you.

## Available Platforms

Conar meets you wherever you work. Use the **web app** directly in your browser at [conar.app](https://conar.app) — no installation required. Download the **desktop app** for [macOS, Windows, or Linux](https://conar.app/download) to run a local proxy and connect to databases on your own machine or private network. Install the **CLI** via npm (`npm install -g conar`) for terminal-based workflows and scripting.
