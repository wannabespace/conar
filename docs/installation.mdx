Tamery is available in three forms: a web app you can use instantly in your browser, a native desktop app for macOS, Windows, and Linux, and a command-line interface installable via npm. Choose the option that best fits your workflow — or use several together, since all three share the same account and cloud-synced connections.

<Note>
  The desktop app is the recommended option for most users. It runs a local proxy on your machine, which makes it straightforward to connect to databases on localhost or within a private network without any extra configuration.
</Note>

## Web App

No installation is needed to use Tamery on the web. Open [tamery.app](https://tamery.app) in any modern browser, sign in with your email or OAuth provider, and you're ready to go. The web app gives you access to every feature — Table Browser, SQL Editor, Schema Visualizer, and AI assistant — from wherever you have internet access.

<Tip>
  You can use the web app to connect to local databases by pairing it with the CLI's `tamery proxy` command. Start the proxy locally with `tamery proxy`, then add a connection in the web app pointing to the proxy address. See the [CLI overview](/cli/overview) for details.
</Tip>

## Desktop App

The Tamery desktop app runs a local proxy automatically, making local and private-network database connections seamless. Download it from [tamery.app/download](https://tamery.app/download) for your platform.

<Tabs>
  <Tab title="macOS">
    **Option 1 — Homebrew (recommended)**

    If you have [Homebrew](https://brew.sh) installed, you can install Tamery with a single command:

    ```bash
    brew install --cask tamery
    ```

    Homebrew will handle downloading, installing, and keeping Tamery up to date.

    **Option 2 — Direct download**

    1. Go to [tamery.app/download](https://tamery.app/download).
    2. Download the macOS installer (`.dmg`) for Apple Silicon or Intel.
    3. Open the downloaded file and drag **Tamery** into your **Applications** folder.
    4. Launch Tamery from Applications or Spotlight.
    5. Sign in with your Tamery account to get started.

    Tamery runs on macOS 12 Monterey and later, on both Apple Silicon and Intel.
  </Tab>
  <Tab title="Windows">
    1. Go to [tamery.app/download](https://tamery.app/download).
    2. Download the Windows installer (`.exe`).
    3. Run the installer and follow the on-screen prompts.
    4. Once installation completes, launch Tamery from the Start menu or your desktop shortcut.
    5. Sign in with your Tamery account to get started.

    Tamery runs on Windows 10 and Windows 11 (64-bit).
  </Tab>
  <Tab title="Linux">
    **Arch Linux (AUR)**

    If you're on Arch Linux or an Arch-based distribution, you can install Tamery from the Arch User Repository using your preferred AUR helper:

    ```bash
    # Using yay
    yay -S tamery-bin
    
    # Using paru
    paru -S tamery-bin
    ```

    After installation, launch Tamery from your application menu or by running `tamery-app` in your terminal.

    **Debian / Ubuntu and derivatives**

    A `.deb` package is available for Debian-based distributions. Download it from [tamery.app/download](https://tamery.app/download) and install it with:

    ```bash
    sudo dpkg -i tamery_*.deb
    ```

    **RPM-based distributions (Fedora, RHEL, openSUSE)**

    An `.rpm` package is available for RPM-based distributions. Download it from [tamery.app/download](https://tamery.app/download) and install it with:

    ```bash
    sudo rpm -i tamery_*.rpm
    # or using dnf
    sudo dnf install ./tamery_*.rpm
    ```

    **AppImage (universal Linux)**

    An AppImage is available for any Linux distribution. Download it from [tamery.app/download](https://tamery.app/download), make it executable, and run it:

    ```bash
    chmod +x Tamery-*.AppImage
    ./Tamery-*.AppImage
    ```
  </Tab>
</Tabs>

## CLI

The Tamery CLI lets you interact with Tamery from your terminal, run a local proxy for web app connections, and integrate Tamery into scripts and automated workflows.

**Requirements:** Node.js version 20 or higher. Check your version with `node --version`.

Install the CLI globally via npm:

```bash
npm install -g tamery
```

Once installed, authenticate by running:

```bash
tamery login
```

This opens your default browser and takes you through the Tamery sign-in flow. Once you've signed in, your credentials are saved locally and the CLI is ready to use.

For a full list of available commands and options, see the [CLI overview](/cli/overview).
