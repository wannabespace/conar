The Tamery CLI brings your database workflow into the terminal. Use it to run SQL queries against any of your saved connections, execute SQL files as part of scripts or CI pipelines, and bridge the Tamery web app to databases that live on your local network or behind a firewall. If you prefer working without leaving your editor — or need to automate database tasks — the CLI is your starting point.

## Installation

The CLI is published as the `tamery` package on npm. Install it globally so the `tamery` command is available anywhere on your system.

```bash
npm install -g tamery
```

**Node.js 20 or later is required.** If you're on an older version, update Node.js before installing.

Once installed, the CLI checks for updates automatically and notifies you in the terminal whenever a newer version is available, so you're always one command away from staying current.

## Authentication

Before running any commands, sign in to your Tamery account:

```bash
tamery login
```

This opens a browser window where you complete the sign-in flow securely. Once authenticated, your session is saved locally and reused across commands.

If your browser doesn't open automatically — for example, when working over SSH — pass the `--no-open` flag and the CLI will print a URL you can copy and open manually:

```bash
tamery login --no-open
```

You have 5 minutes to complete the sign-in flow before the session times out. To check who you're currently signed in as, run `tamery whoami`.

## Using the Proxy

The Tamery web app at [tamery.app](https://tamery.app) connects to your databases directly from the browser. If your database is on a local machine, a private network, or otherwise not reachable from the internet, that direct connection won't work on its own.

The `tamery proxy` command solves this by starting a lightweight local server that acts as a bridge. The web app routes queries through your machine, meaning your database credentials and traffic never leave your local environment. Run it in a terminal window while you're using the web app, and stop it with **Ctrl\+C** when you're done.

```bash
tamery proxy
```
